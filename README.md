# Md Raiyaan Tarique Portfolio

Immersive portfolio built with Vite, React, Tailwind CSS, Framer Motion, and Three.js.

## Local Development

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Quality Checks

- `npm run build`
- `npm run lint`
- `npm run typecheck`

## Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Three.js
